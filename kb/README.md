# База знаний (Knowledge Base)

Положите сюда ваши документы в формате **.md** или **.txt**. Поддерживаются вложенные папки.

После добавления или изменения файлов обязательно пересоберите индекс:

- Локально: `make index` или `./build_index.sh`
- В Docker: `make index-docker` (см. README в корне проекта)
